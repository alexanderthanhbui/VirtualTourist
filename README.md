# VirtualTourist

App that allows users to tour the world without leaving the comforts of their couch. This app allows you to drop pins on a map and pull up Flickr images associated with that location. Locations and images are stored using Core Data.

# Using it

Start the app, press and hold and location on the map to see images of that location.
